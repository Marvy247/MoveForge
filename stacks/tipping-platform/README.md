# Tipping Platform - Content Creator Support System

Enable content creators to receive tips and subscriptions directly from supporters.

## Features

- **Creator Profiles**: Set up profile with name and bio
- **One-Time Tips**: Send tips with optional messages
- **Subscriptions**: Recurring support for creators
- **Supporter Stats**: Track tipping activity and subscriptions
- **Platform Fees**: 2.5% fee supports platform development

## Key Functions

### create-profile
Set up creator profile with display name and bio.

### send-tip
Send a one-time tip to a creator with optional message.

### subscribe / unsubscribe
Start or end recurring subscription to a creator.

## Use Cases

- Video content creators
- Writers and bloggers
- Artists and musicians
- Podcast hosts
- Open source developers
